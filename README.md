# 🌍 Wanderlust – Travel Listing Web App

🚀 A full-stack travel listing platform where users can explore, create, and review beautiful destinations around the world.

---

## ✨ Features

🔐 **Authentication & Authorization**

* User signup & login system
* Secure sessions & protected routes

🏡 **Listings Management**

* Create, edit, and delete travel listings
* Add title, description, price, and location

🖼️ **Image Upload**

* Upload and store images using Cloudinary

⭐ **Reviews & Ratings**

* Users can add reviews and ratings
* Only authors can edit/delete their reviews

📍 **Map Integration**

* View listing locations using interactive maps

📱 **Responsive Design**

* Fully responsive UI using Bootstrap

---

## 🛠️ Tech Stack

**Frontend:**

* EJS / HTML / CSS / Bootstrap

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB (Mongoose)

**Other Tools:**

* Cloudinary (Image Storage)
* Map API (Location Services)
* Passport.js (Authentication)

## ⚙️ Installation & Setup

1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/wanderlust.git
```

2️⃣ Install dependencies

```bash
npm install
```

3️⃣ Add environment variables
Create a `.env` file:

```
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
DB_URL=your_mongodb_url
SESSION_SECRET=your_secret
```

4️⃣ Run the server

```bash
node app.js
```

5️⃣ Open in browser

```
http://localhost:3000
```
